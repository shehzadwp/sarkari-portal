# 🇵🇰 Sarkari Portal — Pakistan Government Information Website

A complete **WordPress-based information portal** for Pakistani citizens — covering government jobs, tests, NADRA, FBR, gold & currency rates, utility bills, government programs, and much more. Built with Elementor and Astra theme.

## ✅ Features

- 💼 **Government Jobs** — Latest Sarkari jobs updated regularly
- 📝 **Tests & Exams** — NTS, PPSC, FPSC, CSS test information
- 🪪 **NADRA Services** — CNIC, passport, birth certificate guides
- 🏦 **FBR & Tax** — Tax filing, NTN registration guides
- 💰 **Gold & Currency Rates** — Live gold price & PKR exchange rates
- 🧾 **Bills Check** — WAPDA, SNGPL, PTCL bill checking guides
- 🏛️ **Government Programs** — Benazir Income Support, Ehsaas & more
- 📱 **Fully Mobile Responsive** — Works perfectly on all devices
- ⚡ **Fast Loading** — Optimized for speed and performance
- 🔍 **SEO Friendly** — Proper structure for search engines
